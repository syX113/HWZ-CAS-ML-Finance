# HWZ-CAS-ML-Finance Repository

## Overview
Welcome to the HWZ-CAS-ML-Finance Repository. This repository is dedicated to providing resources, exercises, and real-world applications for students enrolled in the CAS Machine Learning course at HWZ. It aims to bridge theoretical knowledge with practical insights in the domain of finance.

## Setup Instructions

1. **Clone Repository**: Clone this repository to your local environment.
2. **Virtual Environment**: Set up a new Python virtual environment (venv) to manage dependencies.
3. **Install Dependencies**: Install necessary packages and dependencies as outlined in the `requirements.txt` file.

## Course Blocks

### Block 1: Fundamentals of Financial Data Analysis

**Exercises:**
- Establishing the working environment.
- Loading and visualizing asset data using Python.

**Use-Case:**
- Calculation and visualization of the Efficient Frontier using Python.

### Block 2: Predictive Modeling in Finance

**Exercises:**
- Implementing linear regression in Python with Scikit-Learn for stock price prediction.
- Developing and evaluating a supervised model (linear or logistic regression with scikit-learn).

**Use-Case:**
- Developing a credit risk assessment model using logistic regression in Python.

### Block 3: Advanced Machine Learning Techniques

**Exercises:**
- Performing train/validation/test splits.
- Constructing a basic Neural Network using TensorFlow/Keras.
- Analyzing overfitting/underfitting in trained networks.

**Use-Case:**
- Building an LSTM or RNN model for fraud detection in financial transactions.

### Block 4: Time Series Analysis in Finance

**Exercises:**
- Loading, visualizing, resampling, and interpolating time series data using Pandas/Matplotlib.
- Applying AR, MA, and ARIMA models to financial data.

**Use-Case:**
- Implementing Algorithmic Trading strategies with ARIMA models.
